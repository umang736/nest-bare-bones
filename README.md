## SetUp
Download Nodejs v14+ from https://nodejs.org/en/download/ or install node version manager from https://github.com/coreybutler/nvm-windows/releases
and then install node v14+.
This project was generated with node v14.18.1 and [Nest CLI](https://docs.nestjs.com/cli/overview) version 9.0.0

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
